ELSA: Exact Linear-Scan Attention for Fast and Memory-Light Vision Transformers

Chih-Chung Hsu, Xin-Di Ma, Wo-Ting Liao, Chia-Ming Lee
National Yang Ming Chiao Tung University

CVPR 2026

Paper Code

ELSA reformulates the online softmax recurrence into a parallel prefix scan over an associative monoid of triples $(m, S, W)$[cite: 7, 38]. This reduces parallel depth from $O(n)$ to $O(log\ n)$ while maintaining $O(n)$ memory and exact softmax semantics[cite: 7, 36, 114].

Abstract

Existing attention accelerators often trade exact softmax semantics, depend on fused Tensor Core kernels, or suffer from sequential depth that limits FP32 throughput on long sequences[cite: 5]. We present ELSA, an algorithmic reformulation of softmax attention that:

  • Preserves exactness: Maintains standard softmax semantics in real arithmetic and provides a provable FP32 relative error bound[cite: 6].
  • Parallelizes online softmax: Casts the update into a prefix scan over an associative monoid, yielding $O(n)$ extra memory and $O(log\ n)$ parallel depth[cite: 7].
  • Hardware-agnostic: Independent of Tensor Cores and implemented in Triton and CUDA C++ for cross-platform efficiency[cite: 8].

ELSA improves FP32 inference latency over memory-efficient SDPA by 1.3-3.5x on NVIDIA A100 and Jetson TX2 without requiring retraining[cite: 10, 41].

Key Contributions

Monoid Formulation: We prove that the online softmax recurrence admits an associative binary operation on state triples $(m, S, W) \in \mathbb{R} \times \mathbb{R} \times \mathbb{R}^{d_v}$[cite: 38]. This ensures numerical stability for million-token sequences with a formal FP32 error bound[cite: 38].

Two-Level Scan Architecture: We implement a high-throughput two-level scan (intra-block Hillis-Steele and inter-block Blelloch) tailored for softmax attention[cite: 9, 40].

Training-Free Deployment: ELSA serves as a drop-in replacement for pretrained foundation models like CLIP and SAM, enabling immediate acceleration without fine-tuning[cite: 39, 52].
