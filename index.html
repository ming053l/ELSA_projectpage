TL;DR

Standard attention is slow, and FlashAttention (FA) is strictly bound to specialized hardware (Tensor Cores). ELSA breaks this limitation by turning attention into a "Parallel Scan" problem. It achieves O(log n) parallel depth and O(n) memory on any hardware—from data-center GPUs to embedded devices—without losing precision or requiring zero retraining. Drop it into CLIP or SAM and get up to 3.5x speedup on FP32.

Motivation: Beyond Sequential Bottlenecks

經典的影像處理技術如 Integral Image (積分圖) 展現了如何透過前綴和 (Prefix Sum) 將區域運算優化至 $O(1)$。然而，現代 Transformer 中的 Softmax Attention 卻因為指數運算與歸一化項的相互依賴，陷入了 Sequential Attention 的瓶頸。

現有的加速方案如 FlashAttention 強度依賴於硬體底層的 Tensor Core 融合，導致在 FP32 高精度運算 或 非頂級 GPU (如 AMD, Jetson) 上表現受限。ELSA 的核心突破在於：

  • 從 Sequential 到 Parallel： 我們將 Online Softmax 的遞迴邏輯重新表述為一個結合律么半群 (Associative Monoid) 的掃描問題。
  • 硬體解耦： 借鑒 Blelloch Scan 的思想，ELSA 不再需要特定的硬體指令，即可在任何 GPU 上實現並行化。
  • 無損精度： 不同於線性注意力 (Linear Attention) 的近似，ELSA 提供與標準 Softmax 完全一致的結果，卻擁有 $O(\log n)$ 的並行深度。

Abstract

Existing attention accelerators often trade exact softmax semantics or suffer from sequential depth that limits throughput. We present ELSA, an algorithmic reformulation of softmax attention that preserves exact semantics in real arithmetic. By casting the online softmax update into a prefix scan over an associative monoid of triples $(m, S, W)$, ELSA achieves $O(n)$ extra memory and $O(\log n)$ parallel depth. Implemented in Triton and CUDA C++, it improves FP32 inference latency over memory-efficient SDPA by 1.3–3.5x as a drop-in replacement with no retraining required.

Method: Parallel Monoid Scan

ELSA transforms the sequential online softmax recurrence into a fully parallelizable computation. We prove that attention states $(m, S, W)$ form an associative monoid.

We utilize a two-level scan architecture: intra-block Hillis-Steele and inter-block Blelloch. This reduces the critical path to $O(\log n)$, which is particularly effective for FP32 where Tensor Core fusion is not applicable.
